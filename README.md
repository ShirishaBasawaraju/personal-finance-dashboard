## Personal Finance Dashboard

A responsive web application to track income, expenses, and balance with real-time calculations, analytics, and export features.

## Live Demo

     [https://shirishabasawaraju.github.io/personal-finance-dashboard/]

##  Features

* Add income and expense transactions
* Real-time balance calculation
* Search and filter transactions
* Custom animated dropdown UI
* Pie chart for income vs expense overview
* Export data as PDF report
* Data persistence using LocalStorage

##  Tech Stack

* HTML5
* CSS3 (Modern UI + Animations)
* JavaScript (DOM Manipulation)
* html2pdf.js
